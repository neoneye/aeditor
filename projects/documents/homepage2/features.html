Feature List

AEditor has some unique features which may interest you.

  • Nice Ruby Lexer
    • It deals with multiline literals, multiline comments, escape sequences, interpolated code within strings, regexp, bad ruby constructions. It can even deal with multiple heredocs on the same line!
  • Script With Ruby
    • The editor can be configured + scripted in the programming language Ruby.
  • Quick Theme Switching
    • Its fast to change color theme.
  • Platform independent
    • Due to fox toolkit this application runs on Unix, Mac and Windows.
  • Multiple Buffers
    • Its quick to switch to another buffer.
  • Multilevel Undo + Redo
    • If you do something undesired then it can be undone/redone.
  • Macroes
    • For complex editing you can record a sequence of actions and play them over and over.
  • Nice C/C++ Lexer
    • Inline assember code can be colored.
  • Bookmarks
    • You can place bookmarks which you can get back to later.
  • Fullscreen / Window Mode
    • Fullscreen doesn't work correct on Windows.
  • Theme Designer
    • A new coloring theme can quickly be build with instant feedback.
